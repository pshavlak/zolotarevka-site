### 3. Созданные файлы (10 шт.)

#### CSS (`site/css/style.css`)
- Природная зеленая цветовая гамма (--color-primary: #2d6a4f)
- CSS-переменные для единого стиля
- Адаптивный дизайн (3 брейкпоинта: 992px, 768px, 480px)
- Стили для: мега-меню, бенто-сетки, карточек, таблиц, форм, галереи, модальных окон
- Анимация fadeInUp при скролле
- Стили для блоков: гордость села, доска объявлений, профили игроков

#### JavaScript (`site/js/main.js`)
- Мобильное меню (бургер-кнопка)
- Модальные окна (открытие/закрытие)
- Таймер обратного отсчета до следующего матча
- Отправка формы «Предложить новость»
- Скролл-анимации (Intersection Observer)
- Виджет погоды (демо-данные)
